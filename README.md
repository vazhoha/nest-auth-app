## NestJS Back-end App with authentication &amp; authorisation

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger
Once the application is running you can visit http://localhost:3000/api to see the Swagger interface.
